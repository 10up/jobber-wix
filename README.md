# Jobber Wix App

> [!IMPORTANT]

## Overview
This is the code powering the Jobber Wix App.  
This repository is structure according to the [Wix Documentation](https://support.wix.com/en/article/building-an-app-on-wix)


## Setup 🔧

##### Install dependencies:

```console
npm install
```

## Available Scripts

In the project directory, you can run:
